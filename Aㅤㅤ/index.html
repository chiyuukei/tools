Discord ユーザー情報 >⩊<

このツールは、Discord ユーザーからの情報を見つけるのに役立ちます。情報は十分に機密ではありませんが、直接チェックすることなく即座にチェックするのに非常に適しています。

TikTok タイムスタンプと同様に、入力した ID が有効な場合、このツールは通常どおり機能します。一方、入力した ID が関連性がない場合、情報の精度も低下します。

外観の点ではかなり奇妙に見えたり、反応性が低いように見えるかもしれませんが、私にとって最も重要なことは美学ではなく機能と結果であることは問題ではありません。

ああ、最後に最初にアカウントの開発者モードをアクティブにして、ターゲット アカウントの ID を見つけます。
